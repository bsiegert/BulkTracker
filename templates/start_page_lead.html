BulkTracker is a web app to follow bulk build status in pkgsrc, the NetBSD package collection.

Important announcement

This BulkTracker instance is now shut down.

Unfortunately, recent API deprecations on Google App Engine would have required an almost full rewrite of this app. Instead, this time and effort were spent on rewriting it to run stand-alone, on a NetBSD Foundation server.

Please update your bookmarks to the replacement instance at https://releng.netbsd.org/bulktracker/.
