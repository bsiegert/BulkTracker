BulkTracker is a web app to follow bulk build status in pkgsrc, the NetBSD package collection.

Important announcement

This BulkTracker instance will be shut down on January 23, 2024.

Unfortunately, recent API deprecations on Google App Engine leave no other choice. Please update your bookmarks to the replacement instance at https://releng.netbsd.org/bulktracker/.
